# WooRAG Support

WooRAG Support is a Retrieval-Augmented Generation (RAG) based AI customer support agent
designed for WooCommerce stores.

## Features
- Answers customer queries using store documents (policies, FAQs, product info)
- Order status checking via WooCommerce API
- Source-based and grounded responses
- Reduces human support workload

## Tech Stack
- n8n
- Large Language Model (LLM)
- Vector Database (Chroma/Pinecone)
- WooCommerce REST API

## Use Cases
- Order tracking
- Return & refund queries
- Shipping and delivery questions
- Product information support
